pacing.impl.defaults.mock_audio

Mock audio provider for testing and demonstration.

This implementation generates synthetic audio or plays back pre-scripted scenarios without requiring actual audio hardware.

class pacing.impl.defaults.mock_audio.MockAudioProvider(sample_rate: int = 16000, chunk_duration_ms: int = 100, total_duration_sec: float = 60.0)[source]

Mock audio provider that generates synthetic audio signals.

This implementation is useful for: - Testing the platform without audio hardware - Demonstrating the system in controlled scenarios - Development and CI/CD pipelines

The mock provider can: - Generate silence - Generate synthetic tones - Simulate realistic audio chunk timing
